# Dependancies

- [NodeJS](https://nodejs.org/)
- [couchDB](http://couchdb.apache.org/)


# What does it do?
This will scrape the data from http://www.sismologia.cl/ in a one off process and store it in the couchDB you configure it with.  See config.json
